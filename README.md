# מערכת לניהול פוסט פרודקשיין

מערכת ווב לניהול פרויקטים בפוסט פרודקשיין.

המערכת תומכת בחמישה תחומי עבודה:
1. עריכה
2. עיצוב
3. אנימציה
4. תלת מימד
5. סאונד

השלבים יכולים להתבצע בסדר לינארי או במקביל, לפי תלויות, אישורים, קבצים זמינים, גרסאות נעולות ויחידות עבודה מאושרות.

המערכת צריכה לכלול:
- דשבורד פרויקטים
- מסך פרויקט
- מפת שלבים מקבילית
- ניהול משימות
- ניהול תלויות
- ניהול גרסאות
- נעילה חלקית לפי סצנות, שוטים או מקטעים
- הערות ואישורים
- הרשאות משתמשים
- ממשק בעברית מלאה, ימין לשמאל
